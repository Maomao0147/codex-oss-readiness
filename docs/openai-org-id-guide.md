# OpenAI Organization ID 查找指南

Codex for OSS 申请会要求 OpenAI Organization ID。这个 ID 应该来自你的 OpenAI Platform 账号，而不是 GitHub。

## 查找步骤

1. 打开 OpenAI Platform：<https://platform.openai.com/settings/organization/general>
2. 登录与 ChatGPT 账号相关的 OpenAI 账号。
3. 在 Organization settings / General 页面查找 Organization ID。
4. 把 ID 填到申请表和本地申请工作簿中。

## 记录位置

在本地填写：

- [`application-workbook.md`](application-workbook.md)
- [`week-by-week-tracker.md`](week-by-week-tracker.md)
- copied private file: `private/application-info.private.md`

不要把不想公开的账号信息提交到 public 仓库。如果这个仓库会公开，建议只写：

```text
OpenAI Organization ID: recorded privately
```

然后在真正提交申请表时填写完整 ID。

## 提交前检查

- [ ] 使用的是你自己的 OpenAI 账号。
- [ ] ChatGPT 账号有效。
- [ ] Organization ID 来自 OpenAI Platform，而不是项目名或邮箱。
- [ ] 没有把敏感账号信息公开提交到 GitHub。
